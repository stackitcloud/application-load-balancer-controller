# Application load balancer controller

The STACKIT Application Load Balancer Controller (ALBC) exposes HTTP/HTTPS applications by provisioning and configuring managed STACKIT Application Load Balancers based on Kubernetes resources. Currently, the controller supports the [Ingress API](https://kubernetes.io/docs/concepts/services-networking/ingress/).

This project is currently in alpha stage.
